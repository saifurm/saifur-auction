# saifur-auction

Simple static webpage for Saifur Auction. It shows a friendly greeting and can be deployed anywhere that serves static files (Netlify, GitHub Pages, etc.).

## Local preview

```bash
python -m http.server 4173
# visit http://localhost:4173
```

## Deploy

Upload `index.html` (and any other future assets) to your preferred static host or push this repo to GitHub Pages / Netlify for a public URL.
